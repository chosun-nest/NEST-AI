# Chatbot

## 사용 기술
- 사용 언어 & 라이브러리 : `JavaScript`, `TypeSript`, `ReactJS` 
- 프레임워크 : `NextJS`
- 스타일링 : CSS in JS (`Tailwind CSS`)
- 라이브러리 : `React-spinners`, `React-markdown`, `OpenAI`

**Open AI API**를 활용하여 질문을 전달하고 응답 받는 기능 구현
